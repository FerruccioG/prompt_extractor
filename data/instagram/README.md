
# Instagram Data Artifacts — Pipeline Outputs

## Overview

This directory contains all **Instagram-specific artifacts** generated by the pipeline.

It represents the transformation from raw scraped content into structured, processable prompt data.

---

## 📁 Directory Structure

```text
data/instagram/
├── html/                         # Raw HTML snapshots of posts
├── screenshots/                  # Captured post images (per slide)
├── results.jsonl                 # Scraping metadata per post
├── ocr_raw.jsonl                 # Raw OCR text per image
├── ocr_grouped.jsonl             # OCR text grouped per post
├── text_manipulator_input.jsonl  # Cleaned text ready for semantic processing
