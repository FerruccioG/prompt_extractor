# BOOTSTRAP.template.md

## Purpose

Initial setup guide for establishing agent identity and user context in a fresh OpenClaw workspace.

---

## Steps

1. Define agent identity (name, role, behavior)
2. Define user profile (name, preferences, context)
3. Establish behavioral rules and boundaries
4. Initialize memory files if required

---

## Notes

- This file is used only during first-time setup
- It should not be required in stable or production environments
- Safe to remove after initialization is complete
